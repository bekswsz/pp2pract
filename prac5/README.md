# Practice5 - Python Regular Expressions (RegEx)

This folder contains Python examples and practical exercises using the `re` module.

Topics covered:

- RegEx Introduction
- Metacharacters (., *, +, ?, ^, $, [], |, (), \)
- Special Sequences (\d, \w, \s, \D, \W, \S)
- Quantifiers ({n}, {n,}, {n,m})
- re.search()
- re.findall()
- re.split()
- re.sub()
- re.match()
- Flags (IGNORECASE, MULTILINE)

Practical Task:
- Receipt parsing using raw.txt
- Extracting prices
- Extracting product names
- Calculating total
- Extracting date and time
- Detecting payment method
- Creating structured JSON output

This practice demonstrates real-world text parsing using regular expressions in Python.
